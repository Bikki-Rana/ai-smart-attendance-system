## 🚀 Overview
An intelligent attendance system that leverages **AI and Computer Vision** to automate attendance using **face recognition with liveness detection**, ensuring high security and accuracy.

---

## ✨ Features
- 🎯 Face Recognition-based Attendance  
- 🛡️ Liveness Detection (Anti-spoofing security)  
- ⚡ Fast, automated & contactless system  
- 📊 Reliable and accurate data tracking

## ⚙️ How It Works
1. 📷 Capture face through camera  
2. 🔍 Detect and recognize user  
3. 🧠 Perform liveness check (real vs fake)  
4. ✅ Automatically mark attendance
